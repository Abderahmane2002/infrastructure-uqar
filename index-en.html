Modeling Chloride Transport

2024 - 2025    1 student, UQAR/ENSAM supervision

This project aims to simulate chloride ion penetration in concrete using 1D and 2D numerical models. It is based on diffusion equations coupled with moisture and integrates capillarity effects.

Scientific Approach

The modeling relies on numerical methods (FEM/FDM) and uses experimental data for calibration. An extension to a real case (bridge in Rimouski) is planned.

The main objective is to develop a reliable simulation tool to predict chloride penetration in concrete structures exposed to aggressive environments, such as marine areas or de-icing salt zones.

  • September 2024: Project start and literature review
  • October 2024: 1D modeling and validation
  • December 2024: Extension to a 2D model
  • January 2025: Integration of experimental data
  • April 2025: Application to the Rimouski bridge case

Expected results include simulated chloride concentration profiles, validation by experimental measurements, and 2D visualizations of contamination progression. A prototype user interface for engineers is also planned.
